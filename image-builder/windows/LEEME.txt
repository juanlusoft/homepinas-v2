DESCRIPCION
═══════════

Esta aplicacion crea imagenes personalizadas de Raspberry Pi OS con HomePiNAS
preconfigurado para instalacion automatica en el primer arranque.

Caracteristicas:
  • Interfaz grafica profesional (no requiere conocimientos tecnicos)
  • Descompresion automatica de imagenes .xz y .zip
  • Descarga automatica de 7-Zip si es necesario
  • SSH habilitado automaticamente
  • Instalacion completamente desatendida


INICIO RAPIDO
═════════════

1. Descarga Raspberry Pi OS Lite (64-bit) desde:
   https://www.raspberrypi.com/software/operating-systems/

2. Ejecuta "HomePiNAS-ImageBuilder.bat" (doble clic)
   → Se solicitaran permisos de Administrador automaticamente

3. Selecciona la imagen descargada usando el boton "Examinar..."

4. Pulsa "CREAR IMAGEN HOMEPINAS" y espera

5. Graba la imagen resultante (*-homepinas.img) en una tarjeta SD con:
   - Raspberry Pi Imager: https://www.raspberrypi.com/software/
   - balenaEtcher: https://www.balena.io/etcher/

6. Inserta la SD en tu Raspberry Pi y enciendela

7. HomePiNAS se instalara automaticamente (5-15 minutos)

8. Accede al dashboard: https://<ip-de-tu-raspberry>:3001


REQUISITOS DEL SISTEMA
══════════════════════

  • Windows 10 o Windows 11
  • Permisos de Administrador
  • Conexion a internet (para descargar 7-Zip si no esta instalado)
  • 8 GB de espacio libre (para la imagen descomprimida)


ARCHIVOS INCLUIDOS
══════════════════

  HomePiNAS-ImageBuilder.bat   → Lanzador principal (ejecutar este)
  HomePiNAS-ImageBuilder.ps1   → Aplicacion GUI de PowerShell
  Build-Executable.ps1         → Compilador a .exe (opcional)
  LEEME.txt                    → Este archivo


CREAR EJECUTABLE STANDALONE (OPCIONAL)
══════════════════════════════════════

Para crear un archivo .exe completamente independiente:

1. Abre PowerShell como Administrador

2. Navega a esta carpeta:
   cd "ruta\a\image-builder\windows"

3. Ejecuta el compilador:
   .\Build-Executable.ps1

4. Se creara "HomePiNAS-ImageBuilder.exe"
   Este archivo es portable y no requiere archivos adicionales.


SOLUCION DE PROBLEMAS
═════════════════════

Error: "No se encontro la particion de arranque"
  → Asegurate de usar una imagen oficial de Raspberry Pi OS
  → La imagen debe estar completamente descomprimida (.img)

Error: "Se requieren permisos de Administrador"
  → Haz clic derecho en el .bat y selecciona "Ejecutar como administrador"

La instalacion automatica no arranca:
  → Conecta un monitor para ver el progreso
  → Los logs estan en: /boot/firmware/homepinas-install.log

Instalacion manual por SSH:
  → Usuario: pi (o el que configures en RPi Imager)
  → Ejecuta:
     curl -fsSL https://raw.githubusercontent.com/juanlusoft/homepinas-v2/main/install.sh | sudo bash
